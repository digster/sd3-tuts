Recipe 08 — Depth & Render State

One depth flag and one depth clear turn recipe 07's scramble into a solid cube. The vehicle is bgfx's uint64_t state bitmask — a different philosophy from the 3D track's baked pipelines.

What you'll learn

  • bgfx render state as a per-draw uint64_t bitmask, reset every submit.
  • The write masks (WRITE_RGB, WRITE_A, WRITE_Z) and what disabling each does.
  • Depth testing (DEPTH_TEST_LESS) and clearing the depth buffer (BGFX_CLEAR_DEPTH).
  • Back-face culling (CULL_CW / CULL_CCW) and the convenience BGFX_STATE_DEFAULT.

Prerequisites

The broken cube from recipe 07. This recipe is the smallest possible diff that fixes it — same geometry, same shaders, three changed lines.

Walkthrough

State is data, set per draw

In the 3D track, blend mode, depth mode and cull mode were frozen into an immutable SDL_GPUGraphicsPipeline at creation. bgfx inverts that: render state is a single uint64_t of OR-ed flags you hand to bgfx::setState before each submit, and it resets to nothing afterwards. Want two objects with different blend modes? Two setState calls, one pipeline. It's more flexible and far less boilerplate — at the cost of having to remember to set it every time.

The depth buffer

Recipe 07 broke because nothing tracked which fragments were nearer. The depth buffer is that bookkeeping: per pixel, it stores the closest Z seen so far. Two pieces switch it on.

First, clear depth each frame alongside colour — to the far value, 1.0:

bgfx::setViewClear(0, BGFX_CLEAR_COLOR | BGFX_CLEAR_DEPTH, 0x121a30ff, 1.0f, 0);
//                     ^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^   ^^^^ depth clear value

Second, test and write Z in the draw state:

bgfx::setState(BGFX_STATE_WRITE_RGB
             | BGFX_STATE_WRITE_A
             | BGFX_STATE_WRITE_Z            // record this fragment's depth
             | BGFX_STATE_DEPTH_TEST_LESS    // ...only if it's nearer than what's there
             | BGFX_STATE_CULL_CW);          // skip back faces entirely

DEPTH_TEST_LESS is the usual choice: a fragment survives only if its Z is less than the stored value. WRITE_Z then updates the buffer so later fragments test against it. Drop either and depth testing silently does nothing.

Culling — a free ~50% saving

A solid cube never shows the inside of a face, so drawing back faces is wasted work. BGFX_STATE_CULL_CW discards triangles wound clockwise in screen space (our cube's interior faces). If a model vanishes when you add culling, its winding is the opposite — switch to CULL_CCW.

One macro for the common case. BGFX_STATE_DEFAULT bundles WRITE_RGB | WRITE_A | WRITE_Z | DEPTH_TEST_LESS | CULL_CW | MSAA — exactly the opaque-3D state above. Most recipes from here on could just write bgfx::setState(BGFX_STATE_DEFAULT); we spell the flags out once so you know what's inside it.

Complete listing

Identical to recipe 07 except the two highlighted spots. The cube is now solid and opaque as it tumbles. 

/*
 * bgfx/08-depth-and-render-state.cpp — recipe 07's cube, now with depth.
 * Helpers + cube data identical to recipe 07 (elided). Shaders: vs_cubes/fs_cubes.
 */
#define SDL_MAIN_USE_CALLBACKS 1
#include <SDL3/SDL.h>
#include <SDL3/SDL_main.h>
#include <bgfx/bgfx.h>
#include <bgfx/platform.h>
#include <bx/math.h>
#include <cstdint>

static void *nativeWindowHandle(SDL_Window *);
static void *nativeDisplayHandle(SDL_Window *);
static bgfx::ProgramHandle loadProgram(const char *vs, const char *fs);

struct PosColorVertex { float x, y, z; uint32_t abgr; };
static const PosColorVertex kCube[8] = {
    { -1.0f,  1.0f,  1.0f, 0xff000000 }, {  1.0f,  1.0f,  1.0f, 0xff0000ff },
    { -1.0f, -1.0f,  1.0f, 0xff00ff00 }, {  1.0f, -1.0f,  1.0f, 0xff00ffff },
    { -1.0f,  1.0f, -1.0f, 0xffff0000 }, {  1.0f,  1.0f, -1.0f, 0xffff00ff },
    { -1.0f, -1.0f, -1.0f, 0xffffff00 }, {  1.0f, -1.0f, -1.0f, 0xffffffff },
};
static const uint16_t kCubeIdx[36] = {
    0,1,2, 1,3,2,  4,6,5, 5,6,7,  0,2,4, 4,2,6,
    1,5,3, 5,7,3,  0,4,1, 4,5,1,  2,3,6, 6,3,7,
};

struct AppState {
    SDL_Window              *window = nullptr;
    int                      width = 1280, height = 720;
    bgfx::VertexLayout       layout;
    bgfx::VertexBufferHandle vbh = BGFX_INVALID_HANDLE;
    bgfx::IndexBufferHandle  ibh = BGFX_INVALID_HANDLE;
    bgfx::ProgramHandle      program = BGFX_INVALID_HANDLE;
    Uint64                   startNs = 0;
};

static SDL_AppResult fail(const char *what) { SDL_Log("%s: %s", what, SDL_GetError()); return SDL_APP_FAILURE; }

SDL_AppResult SDL_AppInit(void **appstate, int, char *[])
{
    SDL_SetAppMetadata("bgfx Depth", "1.0", "com.example.sdl3.bgfx.depth");
    if (!SDL_Init(SDL_INIT_VIDEO)) return fail("SDL_Init");

    AppState *st = new AppState(); *appstate = st;
    st->window = SDL_CreateWindow("bgfx — Depth & Render State", st->width, st->height,
                                  SDL_WINDOW_RESIZABLE | SDL_WINDOW_HIGH_PIXEL_DENSITY);
    if (!st->window) return fail("SDL_CreateWindow");
    SDL_GetWindowSizeInPixels(st->window, &st->width, &st->height);

    bgfx::renderFrame();
    bgfx::Init init;
    init.type              = bgfx::RendererType::Count;
    init.platformData.nwh  = nativeWindowHandle(st->window);
    init.platformData.ndt  = nativeDisplayHandle(st->window);
    init.resolution.width  = (uint32_t)st->width;
    init.resolution.height = (uint32_t)st->height;
    init.resolution.reset  = BGFX_RESET_VSYNC;
    if (!bgfx::init(init)) return fail("bgfx::init");

    // CHANGED #1: clear the depth buffer to 1.0 (far) every frame.
    bgfx::setViewClear(0, BGFX_CLEAR_COLOR | BGFX_CLEAR_DEPTH, 0x121a30ff, 1.0f, 0);

    st->layout.begin()
        .add(bgfx::Attrib::Position, 3, bgfx::AttribType::Float)
        .add(bgfx::Attrib::Color0,   4, bgfx::AttribType::Uint8, true)
        .end();
    st->vbh     = bgfx::createVertexBuffer(bgfx::makeRef(kCube, sizeof(kCube)), st->layout);
    st->ibh     = bgfx::createIndexBuffer (bgfx::makeRef(kCubeIdx, sizeof(kCubeIdx)));
    st->program = loadProgram("vs_cubes", "fs_cubes");
    if (!bgfx::isValid(st->program)) return fail("loadProgram");

    st->startNs = SDL_GetTicksNS();
    return SDL_APP_CONTINUE;
}

SDL_AppResult SDL_AppEvent(void *appstate, SDL_Event *e)
{
    AppState *st = static_cast<AppState *>(appstate);
    if (e->type == SDL_EVENT_QUIT) return SDL_APP_SUCCESS;
    if (e->type == SDL_EVENT_KEY_DOWN && e->key.scancode == SDL_SCANCODE_ESCAPE) return SDL_APP_SUCCESS;
    if (e->type == SDL_EVENT_WINDOW_PIXEL_SIZE_CHANGED) {
        st->width = e->window.data1; st->height = e->window.data2;
        bgfx::reset((uint32_t)st->width, (uint32_t)st->height, BGFX_RESET_VSYNC);
    }
    return SDL_APP_CONTINUE;
}

SDL_AppResult SDL_AppIterate(void *appstate)
{
    AppState *st = static_cast<AppState *>(appstate);
    float t = (float)(SDL_GetTicksNS() - st->startNs) / 1.0e9f;

    bgfx::setViewRect(0, 0, 0, (uint16_t)st->width, (uint16_t)st->height);

    const bx::Vec3 at  = { 0.0f, 0.0f,  0.0f };
    const bx::Vec3 eye = { 0.0f, 0.0f, -8.0f };
    float view[16], proj[16];
    bx::mtxLookAt(view, eye, at);
    bx::mtxProj(proj, 60.0f, float(st->width)/float(st->height), 0.1f, 100.0f,
                bgfx::getCaps()->homogeneousDepth);
    bgfx::setViewTransform(0, view, proj);

    float model[16];
    bx::mtxRotateXY(model, t * 0.21f, t * 0.37f);
    bgfx::setTransform(model);

    bgfx::setVertexBuffer(0, st->vbh);
    bgfx::setIndexBuffer(st->ibh);

    // CHANGED #2: write + test depth, and cull back faces.
    bgfx::setState(BGFX_STATE_WRITE_RGB | BGFX_STATE_WRITE_A | BGFX_STATE_WRITE_Z
                 | BGFX_STATE_DEPTH_TEST_LESS | BGFX_STATE_CULL_CW);
    bgfx::submit(0, st->program);

    bgfx::frame();
    return SDL_APP_CONTINUE;
}

void SDL_AppQuit(void *appstate, SDL_AppResult)
{
    AppState *st = static_cast<AppState *>(appstate);
    if (!st) return;
    if (bgfx::isValid(st->ibh))     bgfx::destroy(st->ibh);
    if (bgfx::isValid(st->vbh))     bgfx::destroy(st->vbh);
    if (bgfx::isValid(st->program)) bgfx::destroy(st->program);
    bgfx::shutdown();
    SDL_DestroyWindow(st->window);
    delete st;
}

Try it yourself

  1. Swap CULL_CW for CULL_CCW. The cube turns inside-out — you're now keeping the faces you should be hiding. Then remove culling entirely: it looks correct again (depth alone handles it) but draws twice the triangles.
  2. Drop BGFX_STATE_WRITE_Z but keep DEPTH_TEST_LESS. The first frame looks right, then it degrades — nothing updates the buffer to test against.
  3. Remove BGFX_CLEAR_DEPTH from the view clear. Old depth values linger frame to frame and the cube tears as it spins.
  4. Replace the whole setState with bgfx::setState(BGFX_STATE_DEFAULT). Identical result — confirm it by checking the draw still culls and depth-tests.
